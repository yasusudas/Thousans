このプレビューは内容確認用です。貼り付け前に、販売先の規約、ご自身の販売条件、表示事項に合わせて編集してください。

- `product-descriptions.md`: 商品説明テンプレート50本

## サンプル3テンプレ

### 1. BOOTH / 素材販売

タイトル:
素材販売を始めたい人のための素材パックスターター

1行説明:
利用規約と説明文を短時間で整えるための、Markdown / CSV / PDF 形式の実用テンプレートです。

本文:
この商品は、素材販売を始めたい人が販売前に迷いやすい「何を含めるか」「どこまで説明するか」「注意書きをどう書くか」を整理するためのテンプレート集です。空欄を埋めるだけで、商品説明、内容物一覧、提供形式、FAQ、更新履歴、注意書きを揃えられます。

### 2. note / 有料記事

タイトル:
有料記事を初めて出す人のための有料記事スターター

1行説明:
無料部分と有料部分の境目を決めるための、Markdown / CSV / PDF 形式の実用テンプレートです。

本文:
無料部分で読者に伝えること、有料部分に含めること、購入前の注意点、更新履歴を整理するためのテンプレートです。

### 3. ココナラ / コンテンツ販売

タイトル:
コンテンツ販売を始めたい人のためのテンプレート集スターター

1行説明:
購入後の説明不足を減らすための、Markdown / CSV / PDF 形式の実用テンプレートです。

本文:
購入前に伝えるべき内容、含まれるファイル、使い方、問い合わせ条件を整理するためのテンプレートです。

## 使い方

1. 自分の商品に近い販売先のテンプレートを選ぶ。
2. 対象者、成果物、内容物、提供形式を置き換える。
3. FAQと注意書きを販売先ページの設定に合わせる。
4. 公開前に税込価格、支払方法、提供時期、返金可否、販売者情報、問い合わせ先を確認する。

## 注意事項

売上、審査通過、法的適合を保証するものではありません。テンプレートは下書きとして使い、公開前に各販売先の公式情報とご自身の販売条件を確認してください。
